## woowacourse 8기 최종 코딩테스트

### 크리스마스 프로모션 전체 기능구현 목록
- [ ] 식당 예상 방문날짜를 입력받는다.
    - 예외) 1이상 31이하의 숫자가 아닌경우 "[ERROR] 유효하지 않은 날짜입니다. 다시 입력해 주세요." 에러메시지 출력
- [ ] 주문할 메뉴와 개수를 입력받는다. (아래 예외 1~4에서 "[ERROR] 유효하지 않은 주문입니다. 다시 입력해 주세요." 에러메시지 출력한다.)
    - 예외1) 형식에 맞게 입력되지 않은경우
    - 예외2) 없는 메뉴를 입력한 경우 
    - 예외3) 메뉴의 개수가 1 이상이 아닐경우
    - 예외4) 중복 메뉴를 입력한 경우
    - 예외5) 음식의 총합이 20개 이상인 경우
    - 예외6) 음료만 주문 했을 경우
- [ ] 날짜와 메뉴 등을 고려하여 주문에 대한 혜택 계산을 수행한다.
    - 조건) 총 혜택 금액 = 할인 금액의 합계 + 증정 메뉴의 가격 이다. 
    - 조건) 총 주문 금액 10000원 이상부터 이벤트가 적용된다.
    - 조건) 이벤트 배지는 총 혜택 금액이 5천원 이상이면 별, 1만원 이상이면 트리, 2만원 이상이면 산타이다.
- [ ] 예약일에 받을 이벤트 미리보기 기능
    - [ ] 주문 메뉴를 출력한다.
    - [ ] 할인전 총 주문 금액을 출력한다.
    - [ ] 증정메뉴를 출력한다.
      - 조건) 증정이벤트에 해당하지 않는다면 "없음" 이라고 출력한다.
    - [ ] 혜택 내역을 출력한다.
      - 조건) 적용된 이벤트가 하나도 없다면 혜택 내역 "없음"으로 출력한다.
      - 조건) 여러 개의 이벤트가 적용된 경우, 출력 순서는 자유롭게 한다.
    - [ ] 총혜택 금액을 출력한다.
    - [ ] 할인 후 예상 결제 금액을 출력한다.
    - [ ] 12월 이벤트 배지를 출력한다.
      - 조건) 이벤트 배지가 부여되지 않는 경우, "없음"으로 출력한다.
